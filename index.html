Kasane Teto

Um site criado para celebrar a cantora virtual!

Kasane Teto é uma software de cantor virtual criada no 2channel para o Dia da Mentira de 2008. Embora a software tenha sida criada inicialmente como uma farsa e não existisse, mais tarde ela foi produzida e tornado compatível com o software de síntese de voz UTAU, permitindo que ela cantasse.
